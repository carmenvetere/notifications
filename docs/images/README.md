# Screenshots

Drop the README screenshots here. Capture them with the packaged **example
rules** (Developer Tools → Actions → `notification_center.import_rules`) so only
generic placeholder entities appear — **no personal entity or home names**.

Suggested files (referenced from the top-level `README.md`):

| File | What to capture |
|---|---|
| `panel-rules.png` | The **Notifications** panel — the rule list. |
| `panel-edit.png` | A rule open in the editor, showing the live **card + phone-push** preview (right pane). |
| `card.png` | The `notification-center-card` on a dashboard (ideally light + dark). |
| `push.png` | A phone push notification from the HA companion app. |

Keep them reasonably sized (e.g. ≤ 1600px wide, PNG). Once added, uncomment the
image block in `README.md`.
